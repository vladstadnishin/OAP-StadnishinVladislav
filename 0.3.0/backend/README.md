Проєкт реалізує:

підключення SQLite до бекенду;
автоматичне створення бази даних при першому запуску;
ініціалізацію схеми БД;
роботу з кількома взаємопов'язаними сутностями;
CRUD-операції:
Create
Read
Update
Delete;
обробку помилок;
перевірку цілісності даних через FOREIGN KEY, UNIQUE та NOT NULL.

Структура проєкту
src/
├── routes/
├── repositories/
├── db/
│   ├── db.js
│   ├── dbClient.js
│   └── initDb.js
├── app.js
└── server.js

data/
└── app.db
Встановлення та запуск

Встановити залежності:
npm install
Запустити сервер:
npm start

Сервер буде доступний за адресою:
http://localhost:3000

API-ендпоінти

Отримати всі записи
GET /api/<entity>

Отримати запис за ID
GET /api/<entity>/:id

Створити запис
POST /api/<entity>

Оновити запис
PUT /api/<entity>/:id

Видалити запис
DELETE /api/<entity>/:id

База даних

У проєкті використовується SQLite. База даних автоматично створюється у файлі:
data/app.db

Схема містить декілька взаємопов'язаних таблиць із використанням зовнішніх ключів.
